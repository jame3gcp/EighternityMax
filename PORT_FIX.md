# 포트 3001 충돌 해결 방법

## 문제
포트 3001이 이미 사용 중이라는 에러가 발생했습니다:
```
Error: listen EADDRINUSE: address already in use :::3001
```

## 해결 방법

### 방법 1: 기존 프로세스 종료 (권장)

터미널에서 다음 명령어를 실행하세요:

```bash
# 포트 3001을 사용하는 프로세스 찾기
lsof -ti:3001

# 프로세스 종료 (PID는 위 명령어 결과로 나온 숫자)
kill -9 92731

# 또는 한 번에 종료
kill -9 $(lsof -ti:3001)
```

### 방법 2: 다른 포트 사용

백엔드 서버를 다른 포트(예: 3002)에서 실행하려면:

```bash
# 환경 변수로 포트 지정
PORT=3002 npm run dev

# 또는 .env 파일 생성
echo "PORT=3002" > backend/.env
```

그리고 프론트엔드의 `.env` 파일도 수정:
```
VITE_API_URL=http://localhost:3002
```

### 방법 3: 프로세스 확인 및 수동 종료

1. Activity Monitor (활성 상태 보기) 앱 실행
2. "포트" 탭에서 3001 검색
3. 해당 프로세스 선택 후 종료

## 확인

프로세스를 종료한 후:

```bash
# 포트가 비어있는지 확인
lsof -ti:3001

# 아무것도 출력되지 않으면 성공!
```

그 다음 서버를 다시 실행:

```bash
cd backend
npm run dev
```
