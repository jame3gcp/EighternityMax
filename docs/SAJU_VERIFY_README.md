# 사주 로직 검증 (참조 사이트 기준)

참조: [beta-ybz6.onrender.com](https://beta-ybz6.onrender.com/)

## 개요

- **20건 샘플**: `backend/scripts/saju-verify-samples-20.js`에 정의.
- **참조 수집**: Playwright로 참조 사이트에 동일 입력 후 결과 테이블 추출 → `docs/saju-reference-dataset-20.json` 저장.
- **비교 검증**: 우리 `solarToSaju`/`lunarToSaju` + `enrichSajuWithResults` 결과와 참조 데이터를 항목별 비교 → `docs/SAJU_VERIFY_REPORT_20.md` 생성.

## 실행 방법

### 1. 참조 데이터 수집 (선택)

전체 20건 수집(참조 사이트 응답에 따라 5~10분 소요):

```bash
cd backend && npm run saju-collect-reference
```

일부만 수집:

```bash
cd backend && SAMPLE_LIMIT=5 npm run saju-collect-reference
```

최초 실행 시 Chromium 설치:

```bash
cd backend && npx playwright install chromium
```

### 2. 검증 실행

```bash
cd backend && npm run saju-verify-dataset
```

- 성공 시: 종료 코드 0.
- 핵심 항목(간지 4주·오행) 불일치가 있으면 종료 코드 1.
- 상세 리포트: `docs/SAJU_VERIFY_REPORT_20.md`.

## 비교 항목

| 항목 | 설명 | 비고 |
|------|------|------|
| 간지 4주 | 연·월·일·시주 (한/중) | 핵심 |
| 오행 | 목·화·토·금·수 개수 | 핵심 |
| 십성 | 연월일시 천간 십성 | 참조는 천간+지지 십성 행 다수 |
| 12운성 | 연월일시 지지 12운성 | 참조는 봉법/거법 둘 다 표시, 우리는 長生 기준 1종 |
| 십이신살·신살 | 역마/도화/화개 등 | 참조는 항목 수 많음 |
| 대운 | 10단계 간지·십성·신살·12운성 | 기산·순서 차이 가능 |
| 세운 | 연도별 간지 | 기준 연도·열 순서 차이 가능 |
| 월운 | 1~12월 간지 | 당해년·월 기준 차이 가능 |

**핵심 판정**: 간지 4주·오행이 모두 참조와 일치하면 해당 케이스는 “일치”로 간주. 대운/세운/월운·12운성·신살은 참조와 기준·표기가 다를 수 있어 불일치만으로 실패로 보지 않음.

## 알려진 차이

1. **시간 미입력**  
   참조는 시간 없을 때 **0:00(子时)** 로 계산. 검증 시 우리도 비교용으로 `00:00`을 사용해 동일 조건으로 비교.

2. **12운성**  
   참조는 **봉법12운성**·**거법12운성** 두 종류를 표시. 우리는 長生(地支) 기준 한 종류만 사용.  
   비교 시 참조의 “거법12운성” 행과 매핑.

3. **대운·세운·월운**  
   대운 기산(起运) 방식, 세운/월운의 기준 연도·월·열 순서가 참조와 다를 수 있어 간지가 반대 순서나 다른 값으로 나올 수 있음.  
   핵심 판정에는 사용하지 않음.

4. **신살**  
   참조는 천을귀인·월덕귀인·관귀학관 등 항목이 많고, 우리는 재살/지살/겁살·월덕귀인·역마/도화/화개 등 일부만 제공.  
   항목 수·이름 차이로 인한 불일치는 참고용으로만 활용.

## 참조 사이트에 맞춘 로직 변경

- **연·월주(立春·节气)**: 양력 기준 **입춘(立春)換年**·**节气換月** 적용. 입춘일 근사식(공历 2월 3~5일), 절기 경계는 공历 대략일(예: 清明 4/5, 惊蛰 3/5). `solarToSaju`·`lunarToSaju` 모두 변환된 양력일 기준으로 연·월주 보정.
- **시주(时辰)**: 참조와 동일 — 분이 30분 미만이면 해당 정시 구간, 30분 이상이면 다음 시진. 예: 01:00=子, 03:30=寅, 15:45=申, 23:00=亥.
- **월운(流月)**: 1月=子, 2月=亥, …, 12月=丑 순서(참조 테이블 역순 月建) 및 12월만 전년 年干 적용.
- **세운 비교**: 참조 테이블 열 순서가 2031→2022 역순이므로, 비교 스크립트에서 같은 연도끼리 매핑해 비교.

대운 순서(顺/逆)·12운성(封法/格法)·신살 항목 수는 참조와 해석 차이가 있어 현재 로직 유지.
