# 만세력 테스트 비교 결과

참조: korean-lunar-calendar README(KARI), 웹 검색, [beta-ybz6.onrender.com](https://beta-ybz6.onrender.com/)

## 요약

| 구분 | 건수 |
|------|------|
| 일치 | 3 |
| 불일치 | 0 |
| 참조 없음(수동 확인) | 3 |

## 참조 사이트와 수동 비교용 입력

[beta-ybz6.onrender.com](https://beta-ybz6.onrender.com/)에서 아래와 같이 입력한 뒤, 사주 4주(연·월·일·시주)를 우리 결과와 비교하면 됩니다.

| 케이스 | 양력/음력 | 생년월일 | 시간 | 성별 |
|--------|-----------|----------|------|------|
| README-solar-2017 | 양력 | 2017-06-24 | - | 남 |
| README-lunar-1956 | 음력 | 1956-01-21 | - | 여 |
| with-time-1990 | 양력 | 1990-05-15 | 14:30 | 남 |
| with-time-2000 | 양력 | 2000-01-01 | 09:00 | 여 |
| solar-1988-02-17 | 양력 | 1988-02-17 | 08:00 | 남 |
| lunar-with-time | 음력 | 1985-08-15 | 20:00 | 남 |

## 케이스별 결과

### README-solar-2017: korean-lunar-calendar README: 양력 2017-06-24

**입력**
```json
{
  "birthDate": "2017-06-24",
  "birthTime": null,
  "calendarType": "solar",
  "isIntercalation": false,
  "gender": "M"
}
```

**우리 결과**
- 연주(한): 정유년
- 월주(한): 병오월
- 일주(한): 임오일
- 시주(한): -
- 연주(중): 丁酉年
- 월주(중): 丙午月
- 일주(중): 壬午日
- 시주(중): -
- 성별: M
- 음력: 2017-5-1(윤)
- 양력: 2017-6-24
- 오행분포: 목0 화4 토0 금1 수1
- 십성: 연정재 월편재 일비견 시-
- 12운성: 연목욕 월태 일태 시-

**비교**
| 항목 | 우리 | 참조 | 일치 |
|------|------|------|------|
| 음력 | 2017-5-1(윤) | 2017-5-1(윤) | O |
| 간지(한) year | 정유년 | 정유년 | O |
| 간지(한) month | 병오월 | 병오월 | O |
| 간지(한) day | 임오일 | 임오일 | O |
| 간지(중) year | 丁酉年 | 丁酉年 | O |
| 간지(중) month | 丙午月 | 丙午月 | O |
| 간지(중) day | 壬午日 | 壬午日 | O |

**결과**: O 일치
